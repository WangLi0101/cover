### 伪数组包括

- 函数的 `arguments` 对象
- `NodeList` 对象
- 字符串
- `TypedArray` 对象
- `Set` 和 `Map` 结构的 `values` 方法
- `FormData` 对象
- 拥有 `length` 属性和若干索引的对象

### 方法一： `Array.from`

```javascript
console.log(Array.from(Array(100), (x) => 1));
// 转化伪数组
Array.from(ArrayLike);
```

### 方法二： `Array.prototype.slice`

```javascript
console.log(Array.prototype.slice.call(Array(100), 0));
// 转化伪数组
Array.prototype.slice.call(ArrayLike, 0);
```
