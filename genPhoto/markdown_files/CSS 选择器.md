# CSS 选择器

CSS 选择器是用于选择 HTML 元素并应用样式的模式。在这篇文章中，我们将深入探讨各种 CSS 选择器及其用法。

## 1. 基本选择器

### 1.1 元素选择器

元素选择器用于选择所有指定类型的 HTML 元素。例如，选择所有的段落元素：

```css
p {
  color: blue;
}
```

### 1.2 类选择器

类选择器用于选择具有特定类的元素。类选择器以 `.` 开头，后跟类名。例如，选择所有具有 `button` 类的元素：

```css
.button {
  background-color: green;
}
```

### 1.3 ID 选择器

ID 选择器用于选择具有特定 ID 的元素。ID 选择器以 `#` 开头，后跟 ID 名称。注意：ID 在页面中应该是唯一的。

```css
#header {
  background-color: gray;
}
```

## 2. 组合选择器

### 2.1 后代选择器

后代选择器用于选择某个元素内的所有指定后代元素。它由两个或多个选择器组成，之间用空格分隔。

```css
div p {
  color: red;
}
```

### 2.2 子选择器

子选择器用于选择某个元素的直接子元素。它由两个选择器组成，之间用 `>` 分隔。

```css
ul > li {
  list-style-type: none;
}
```

### 2.3 相邻兄弟选择器

相邻兄弟选择器用于选择紧接在另一个元素之后的元素。它由两个选择器组成，之间用 `+` 分隔。

```css
h1 + p {
  margin-top: 0;
}
```

### 2.4 通用兄弟选择器

通用兄弟选择器用于选择在另一个元素之后的所有兄弟元素。它由两个选择器组成，之间用 `~` 分隔。

```css
h1 ~ p {
  color: purple;
}
```

## 3. 属性选择器

属性选择器用于选择具有特定属性的元素。可以根据属性的存在、值、部分值等进行选择。以下是属性选择器的详细用法：

### 3.1 存在属性选择器

选择具有指定属性的所有元素，而不考虑属性的值。

```css
/* 选择所有具有 title 属性的元素 */
[title] {
  border: 1px solid black;
}
```

### 3.2 精确匹配属性选择器

选择具有指定属性和值的元素。

```css
/* 选择所有 type 属性为 text 的 input 元素 */
input[type="text"] {
  border: 1px solid black;
}
```

### 3.3 包含词匹配属性选择器

选择属性值中包含指定词的元素。词是以空格分隔的。

```css
/* 选择所有 class 属性中包含 'button' 词的元素 */
[class~="button"] {
  background-color: yellow;
}
```

### 3.4 开头匹配属性选择器

选择属性值以指定字符串开头的元素。

```css
/* 选择所有 href 属性以 'https' 开头的链接 */
a[href^="https"] {
  color: green;
}
```

### 3.5 结尾匹配属性选择器

选择属性值以指定字符串结尾的元素。

```css
/* 选择所有 href 属性以 '.pdf' 结尾的链接 */
a[href$=".pdf"] {
  color: red;
}
```

### 3.6 子串匹配属性选择器

选择属性值中包含指定子串的元素。

```css
/* 选择所有 href 属性中包含 'example' 的链接 */
a[href*="example"] {
  font-weight: bold;
}
```

## 4. 伪类选择器

伪类选择器用于选择元素的特定状态。常见的伪类包括 `:hover`、`:focus`、`:nth-child()` 等。

```css
a:hover {
  color: orange;
}
```

## 5. 伪元素选择器

伪元素选择器用于选择元素的特定部分。常见的伪元素包括 `::before`、`::after`、`::first-line` 等。

```css
p::first-line {
  font-weight: bold;
}
```

## 6. 组合使用选择器

选择器可以组合使用，以实现更复杂的选择逻辑。例如，选择所有类为 `button` 的元素中，紧接在 `h1` 元素之后的段落：

```css
h1 + p.button {
  color: blue;
}
```

## 结论

CSS 选择器是强大的工具，能够帮助您精确地选择和样式化网页中的元素。通过理解和运用这些选择器，您可以创建出更具表现力和功能性的网页设计。
