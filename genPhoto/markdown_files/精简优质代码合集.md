### JS 如何翻转一个字符串

#### 方法一

```javascript
const reverse = (s) => s.split("").reverse().join("");
```

#### 方法二

```javascript
const reverse = (s) => [...s].reduceRight((acc, cur) => acc + cur);
```

### 如何过滤数组中的 falsy value

```javascript
// falsy 值包括 false、0、""（空字符串）、null、undefined 和 NaN。
const filteredArr = arr.filter(Boolean);
```

### JS 如何生成一个 n 位的随机字符串

```javascript
// 正常生成的字符串类似： 0.123456789abcdefg
// 所以需要截取掉前面的 0.
const random = (n) =>
  Math.random()
    .toString(36)
    .slice(2, 2 + n);
```
