### 1. 使用 `padding` 技巧

这种方法最常见，用 `padding-top` 或 `padding-bottom` 来实现，因为 `padding` 是相对于元素的宽度计算的。

```html
<div class="aspect-ratio-box">
  <div class="content"></div>
</div>
```

```css
.aspect-ratio-box {
  position: relative;
  width: 100%; /* 设置宽度 */
  padding-top: 56.25%; /* 高度是宽度的56.25% (16:9的比例) */
}

.content {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background-color: lightblue; /* 可以添加背景颜色或其他内容 */
}
```

### 2. 使用 `aspect-ratio` 属性

这是较新的方法，在现代浏览器中支持。可以直接使用 `aspect-ratio` 来设置宽高比：

```html
<div class="aspect-ratio-box"></div>
```

```css
.aspect-ratio-box {
  width: 100%; /* 设置宽度 */
  aspect-ratio: 16 / 9; /* 设置16:9的宽高比 */
  background-color: lightcoral; /* 可以添加背景颜色 */
}
```

### 3. 使用 `object-fit` 技巧（适用于图片或视频）

如果要在图片或视频上保持比例，可以使用 `object-fit`：

```html
<img src="image.jpg" class="aspect-ratio-img" />
```

```css
.aspect-ratio-img {
  width: 100%;
  height: 100%;
  object-fit: cover; /* 保持比例并裁剪 */
  aspect-ratio: 16 / 9; /* 直接使用aspect-ratio */
}
```
