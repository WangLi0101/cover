### 什么是安全整数，如何判断一个整数是安全整数

`Number.MAX_SAFE_INTEGER` 静态数据属性表示在 JavaScript 中最大的安全整数（$2^{53} - 1$）。

`Number.isSafeInteger()` 方法确定传递的值是否为安全整数。

```javascript
const x = Number.MAX_SAFE_INTEGER + 1;
const y = Number.MAX_SAFE_INTEGER + 2;

console.log(Number.MAX_SAFE_INTEGER); // 9007199254740991

console.log(x); // 9007199254740992

console.log(y); // 9007199254740992

console.log(x === y); // true

console.log(Number.isSafeInteger(Number.MAX_SAFE_INTEGER)); // true

console.log(Number.isSafeInteger(Number.MAX_SAFE_INTEGER + 1)); // false

console.log(Number.isSafeInteger(x)); // false

console.log(Number.isSafeInteger(y)); // false
```

### 扩展

#### `Number.MAX_SAFE_INTEGER` 和 `Number.MAX_VALUE` 的区别

- `Number.MAX_SAFE_INTEGER` 是 JavaScript 中可以安全表示的最大整数，其值为 $2^{53} - 1$，即 `9007199254740991`。这个值是由于 JavaScript 使用双精度浮点数表示数字，因此超过这个值会出现精度丢失的问题。

- 而 `Number.MAX_VALUE` 则是 JavaScript 中可以表示的最大的正数，其值为 `1.7976931348623157e+308`。这个值是由于 JavaScript 使用双精度浮点数表示数字，因此可以表示的最大整数和最大浮点数是不同的。
