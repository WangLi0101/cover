纯函数是指在相同的输入条件下，总是会返回相同的输出结果，并且没有副作用的函数。具有以下特点：

1. 相同的输入始终返回相同的输出：纯函数的输出结果仅依赖于输入，不受外部状态的影响。
2. 没有副作用：纯函数不会对外部环境造成影响，不会改变状态或进行 I/O 操作。
3. 可缓存性：由于纯函数的输出结果只依赖于输入，可以缓存中间结果来提高性能。

纯函数的特点使得代码更易于理解、测试和维护，有助于减少代码的复杂性。在函数式编程中，纯函数被认为是推崇的编程风格。

下面是一个使用 JavaScript 的纯函数示例：

```javascript
// 纯函数：将传入的数字加1
function addOne(num) {
    return num + 1;
}

// 调用 addOne 函数多次，传入相同的参数会得到相同的结果
console.log(addOne(5)); // 输出 6
console.log(addOne(5)); // 输出 6

// 纯函数：计算两个数字的乘积
function multiply(x, y) {
    return x * y;
}

// 调用 multiply 函数，传入相同的参数会得到相同的结果
console.log(multiply(3, 4)); // 输出 12
console.log(multiply(3, 4)); // 输出 12
```

这两个函数 `addOne` 和 `multiply` 都是纯函数，因为它们只依赖于输入参数并且没有副作用。无论何时调用这两个函数，传入相同的参数都会得到相同的输出结果，不会对外部环境造成影响。
