### 暗黑模式 (Dark Mode)

```html
<div class="container">
  Lorem ipsum dolor sit amet consectetur adipisicing elit. Excepturi laborum
  assumenda neque quam veritatis cumque ipsam corrupti fuga corporis temporibus
  voluptas, est dolores iure asperiores distinctio repudiandae tempora, quos
  labore.
</div>
```

```css
* {
  margin: 0;
  padding: 0;
}

@media (prefers-color-scheme: light) {
  :root {
    --main-bg-color: #f0f0f0;
    --main-text-color: #000;
  }
}

@media (prefers-color-scheme: dark) {
  :root {
    --main-bg-color: black;
    --main-text-color: white;
  }
}

.container {
  width: 100vw;
  height: 100vh;
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}
```
