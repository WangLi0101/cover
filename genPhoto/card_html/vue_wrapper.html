01 / 04 Vue Component Wrapping
Props 穿透

通过 $attrs 透传未声明的属性，合并 props 和 $attrs 一起传递

Template 
<template>
  <!-- 关键：合并 props 和 $attrs -->
  <el-input v-bind="{ ...props, ...$attrs }" />
</template>
Script 
<script setup lang="ts">
import { InputProps } from "element-plus";

// 使用 Partial 获取完整的 TS 类型提示
const props = defineProps<Partial<InputProps>>();
</script>
TIP: v-bind="{ ...props, ...$attrs }" 会将声明的 props 和未声明的 $attrs 一起透传给内部组件
02 / 04 $attrs Passthrough
SLOTS 穿透
方法一：遍历 $slots 
<el-input v-bind="{...props, ...$attrs}">
  <!-- 遍历所有插槽并转发 -->
  <template
    v-for="(index, name) in $slots"
    v-slot:[name]>
    <slot :name="name" />
  </template>
</el-input>
VS
✓ 方法二：h 函数 
<!-- h函数直接传递$slots -->
<component
  :is="h(ElInput, 
    { ...$attrs, ...props, ref: getRefInstance },
    $slots)"
/>
h 函数参数： h(组件, 属性对象, 插槽对象) — 第三个参数直接传 $slots 即可完成所有插槽的穿透
03 / 04 Slots Passthrough
EXPOSE 方法暴露

通过 ref 获取内部组件实例，将其赋值给 vm.exposed 暴露给父组件

完整代码 
<template>
  <component
    :is="h(ElInput, { ...$attrs, ...props, ref: getRefInstance }, $slots)"
  />
</template>

<script setup lang="ts">
import { ElInput, InputInstance, InputProps } from "element-plus";
import { getCurrentInstance, h } from "vue";

const props = defineProps<Partial<InputProps>>();
const vm = getCurrentInstance();

// 关键：将内部组件实例暴露给父组件
const getRefInstance = (inputInstance: InputInstance) => {
  vm.exposed = inputInstance || {};
  vm.exposeProxy = inputInstance || {};
};
</script>
✓ 父组件现在可以通过 ref 调用内部组件的所有方法
