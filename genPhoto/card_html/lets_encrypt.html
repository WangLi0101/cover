INSTALL

安装 Certbot

加载官方客户端工具...

Debian / Ubuntu

APT-GET_
sudo apt update
sudo apt install certbot python3-certbot-nginx

CentOS / RHEL

YUM_PM_
sudo yum install epel-release
sudo yum install certbot python3-certbot-nginx
PAGE_03 // INSTALL CORE_UPDATED
EXECUTE

获取证书

执行自动化配置脚本...

Nginx Plugin

CERTBOT.EXERUNNING 
sudo certbot --nginx

Apache Plugin

CERTBOT.EXERUNNING 
sudo certbot --apache
CAUTION:

跟随 CLI 交互指引：输入邮箱 -> 同意协议(A) -> 选择域名。
