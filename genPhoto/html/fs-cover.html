Node.js
File System

The fs module enables interacting with the file system in a way modeled on standard POSIX functions.

✔ fs.readFile // Read file content
✔ fs.writeFile // Write to file
⚠ fs.unlink // Delete file (Careful!)
ℹ fs.promises // Async/Await API
# Import Module
const fs = require('fs/promises');
# Async Operation
async function main() {
  try {
    const data = await fs.readFile(
      './config.json'
    );
    console.log('Success!');
  } catch (err) {
    console.error(err);
  }
}
